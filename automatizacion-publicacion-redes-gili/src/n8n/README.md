# Apps Script — Scheduling Engine

Este módulo gestiona la lógica de planificación automática de publicaciones.

## Responsabilidades
- Interpretar frecuencias en lenguaje natural
- Calcular fechas válidas según ventana horaria
- Evitar colisiones de publicación
- Mantener consistencia temporal

## Motivo de uso de Apps Script
- Integración nativa con Google Sheets
- Bajo coste operativo
- Alta fiabilidad para procesos recurrentes
